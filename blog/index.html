Getting started to Nephia

2013-09-29 11:32

Install Nephia

$ cpanm Nephia

Create your project

$ nephia-setup YourApp

Up your project

$ cd ./YourApp
$ plackup

Then, you can access to http://nephia.github.io:5000/ .
