News

API full change coming soon

21 Aug. 2013

by ytnobody

Now we're develop new Nephia as "codename Voson".

Nephia contributors was determine development direction of Nephia on some debates.

  • Internal API becomes OOP. We trying to repulse black magic from code as possible.
  • DSL-like interface is still alive on user-side API. I think simplified syntax is good things.
  • Pluggable mechanism will include. This factor is good news for plugin author.
  • Setup will have a pluggable mechanism. This is good news too for plugin author.

In our schedule, this outcome will be released in final week of September 2013.
